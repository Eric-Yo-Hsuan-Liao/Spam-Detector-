# Spam Detector Project
This project uses this dataset: https://www.kaggle.com/datasets/purusinghvi/email-spam-classification-dataset/data
I was not able to have it on github because the file is over 100 mb, but it is easily downloadable using the link above.
## Overview
Using a dataset of 83,446 emails, I created a spam detector using several models including Naive Bayes, SVM, and Logistic Regression. SVM was the best model, with a test accuracy of 98.52% and a f1-score of 0.99.
